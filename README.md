## âœ¨ Introduction

**SNTS'DARK#01** is a sleek, modern, and dark theme for Visual Studio Code, heavily inspired by the beloved **GitHub Dark Mode**. If you're a fan of that clean, high-contrast, and easy-on-the-eyes aesthetic, you'll feel right at home.

This theme is designed to be your daily driver, providing a comfortable and visually appealing coding environment for long hours of work. The color palette is carefully selected to be vibrant and colorful where it matters, without being distracting, all while maintaining a dark and edgy vibe.
